######### Instructions for compiling project #####

- Download the project and unzip project.

- delete the CmakeListCache.txt or any other files named somethin with CmakeList BESIDES the files actually called CmakeList.txt !!!!!!!

- Open project in IDE and run cmake.

- Change the paths in the source file according to where you have put the project.
